# 04_制作物

## このフォルダに入れるもの

実際に**作ったもの（完成品・試作品）**を入れる場所です。

### 具体的には…
- HTML / CSS / JavaScript で作ったWebページ
- 営業デモ用のツール・画面
- チラシ・資料のデータ
- 業務改善のために作ったツール

### フォルダの分け方（例）
```
04_制作物/
├── デモ用Webページ/
├── 営業ツール/
└── 業務改善ツール/
```

### 注意
- 最初は HTML / CSS / JavaScript を中心に作る
- 完成品だけでなく「途中の試作品」も保存してOK
