# mafty-os-v0.3

たける専用AIチーム「マフティー・ナビーユ・エリン」のオペレーティングシステム。  
5体のエージェントと同時にチャットでき、日々の業務・営業・制作を支援するためのローカルダッシュボードです。

---

## 現在できること

| 機能 | 説明 |
|------|------|
| 5体のエージェント横並びチャット | ハサウェイ・ギギ・イラム・ケネス・アナハイムに同時にメッセージを送れる |
| TODAY'S COMMAND | 今日の指令・メモを入力して保存できる |
| PROJECT STATUS | 進行中プロジェクトのステータスを入力・表示できる |
| MAFTY STATUS | チーム全体の状態メモを記録できる |
| GIGI TASK BOARD | ギギ専用のタスク管理ボード。タスクを追加・ステータス切り替え・削除できる |
| タスクのステータス切り替え | 未着手 → 作業中 → 完了 の順にクリックで循環切り替え |
| EXPORT LOG | ダッシュボード・タスク・会話履歴をMarkdown形式でダウンロードできる |
| EXPORT対象選択 | エクスポートするデータをチェックボックスで選択できる |
| localStorage保存 | チャット履歴・ステータス・タスク・EXPORT選択状態を自動保存 |
| 履歴クリア | 各エージェントのチャット履歴を個別にクリアできる |

---

## GIGI TASK BOARD の使い方

1. 画面中央の「GIGI TASK BOARD」パネルでタスクを入力し「追加」ボタンまたは Enter で追加
2. ステータスボタンをクリックすると「未着手 → 作業中 → 完了」と順番に切り替わる
3. 「削除」ボタンでタスクを削除する
4. タスクの内容・ステータスはページを更新しても保持される（localStorage保存）

---

## EXPORT LOG の使い方

1. ヘッダー右端の「EXPORT LOG ▼」ボタンをクリックしてドロップダウンを開く
2. 出力したい項目にチェックを入れる

   | 選択項目 | 内容 |
   |---------|------|
   | ダッシュボード | TODAY'S COMMAND / PROJECT STATUS / MAFTY STATUS |
   | GIGI TASK BOARD | タスク一覧とステータス |
   | ハサウェイ〜アナハイム | 各エージェントとの会話履歴 |

3. 「エクスポート実行」をクリックすると `mafty-log-YYYY-MM-DD-HHMM.md` がダウンロードされる
4. 全項目のチェックを外した状態で実行すると警告が表示されダウンロードしない
5. チェック状態はページ更新後も保持される

## 起動方法

1. VS Code でプロジェクトフォルダを開く
2. `04_制作物/mafty-os-v01/index.html` を開く
3. 右クリック →「Open with Live Server」を選択（または右下の「Go Live」ボタン）
4. ブラウザが自動で起動し、ダッシュボードが表示される

> Live Server 拡張機能がない場合は、VS Code の拡張機能タブで「Live Server」を検索してインストールする。

## 注意事項

- **localStorage保存**のため、ブラウザ・URLの組み合わせ単位でデータが保存される
- ブラウザを変えると履歴は共有されない（Chrome で保存した履歴は Firefox では見えない）
- プライベートモード・シークレットモードでは保存されない場合がある
- Git操作は明示的な指示があるまで行わない
