# 07_サブエージェント

## このフォルダに入れるもの

**AIの自動化・サブエージェント**に関するファイルを入れる場所です。

### サブエージェントって何？
*小学生向け説明：「お使いを頼める小さなAI」のようなものです。メインのAI（Claude）が別のAIに「この仕事やっといて」と頼む仕組みです。*

### 具体的には…
- 自動化タスクの設定ファイル
- サブエージェントへの指示書
- 定期実行するタスクのスクリプト
- エージェントの動作記録・ログ

### 注意
- サブエージェントを動かす前に、必ず許可確認を取る
- 外部サービスへの接続を伴う場合は特に慎重に行う
- 何をするエージェントなのかを必ずメモに残す

---
