# 06_外部連携メモ

## このフォルダに入れるもの

Gmail・Slack・Google など**外部サービスとつなぐときの設定メモ・記録**を入れる場所です。

### 具体的には…
- 連携するサービスの名前・用途のメモ
- 設定手順の記録
- 「どこまで操作させるか」の決定事項
- トラブルが起きたときの対処メモ

### 重要な注意（必ず守ること）
- **パスワード・認証キーは絶対にここに書かない**
  → `.env` という別ファイルで管理する
- 外部連携は最初は**読み取り専用（見るだけ）を原則**にする
- テストは本番アカウントではなく、テスト用アカウントで行う
- 外部連携を行う前に、必ず許可確認を取る

---
*小学生向け説明：GoogleやLINEなど「外のサービス」とつなぐときのメモを入れる引き出しです。パスワードは絶対ここに書かないでください。*
